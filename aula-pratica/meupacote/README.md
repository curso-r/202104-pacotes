O objetivo do pacote é disponibilizar funções para analisar dados do
Brasileirão.

## Installation

And the development version from [GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("beatrizmilz/meupacote")
```

## Exemplo de uso

``` r
library(meupacote)
```
